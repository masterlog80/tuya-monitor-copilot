How to Get Tuya Credentials

  1. Go to iot.tuya.com and sign in.
  2. Create a new project or open an existing one.
  3. Find the Access ID and Access Secret on the project's overview page.
  4. Link your devices to the project under "Devices" → "Link App Account".
  5. Choose one of the two options to identify your account:
    • User Code (recommended) – Open the Tuya Smart or Smart Life app, go to Profile → tap your avatar → User Code. Enter this code in the User Code field above. The User ID will be resolved automatically.
    • User ID – Find your UID under "Devices" → "Link App Account" → your account's UID on iot.tuya.com.
  6. Select the correct region matching your Tuya account region.
